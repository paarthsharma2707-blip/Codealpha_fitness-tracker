# 💪 Fitness Tracker

A modern Fitness Tracker web application built with HTML, CSS, and JavaScript to help users monitor their daily activities, fitness goals, and workout history.

---

## ✨ Features

- 🏃 Log fitness activities
- 📊 Track daily goals
- 👣 Step counter
- 🔥 Calories burned
- ⏱️ Active minutes tracker
- 💧 Water intake progress
- 📅 Weekly activity summary
- 📈 Progress bars
- 📝 Activity history
- 📱 Responsive design
- 🌙 Dark mode support

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- ## 📚 Learning Outcomes

This project demonstrates:

- DOM Manipulation
- Event Handling
- Dynamic UI Updates
- Responsive Web Design
- Progress Tracking
- JavaScript Objects & Arrays
- Interactive Forms
